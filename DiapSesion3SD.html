📋 LEYES BÁSICAS DEL ÁLGEBRA BOOLEANA

🔹 LEYES DE IDENTIDAD

A + 0 = A
A · 1 = A

🔹 LEYES DEL ELEMENTO NULO

A + 1 = 1
A · 0 = 0

🔹 LEYES DE IDEMPOTENCIA

A + A = A
A · A = A

🔹 LEYES DEL COMPLEMENTO

A + Ā = 1
A · Ā = 0

🎮 ¡Practica las leyes básicas!

⚡ LEYES AVANZADAS

🔄 LEYES CONMUTATIVAS

A + B = B + A
A · B = B · A

🔗 LEYES ASOCIATIVAS

(A + B) + C = A + (B + C)
(A · B) · C = A · (B · C)

📊 LEYES DISTRIBUTIVAS

A · (B + C) = A·B + A·C
A + (B · C) = (A+B)·(A+C)

🌪️ LEYES DE ABSORCIÓN

A + (A · B) = A
A · (A + B) = A

🔥 TEOREMAS DE DE MORGAN

Los teoremas más importantes para simplificación de circuitos:

📜 PRIMER TEOREMA

(A + B)̄ = Ā · B̄

"La negación de una suma es igual al producto de las negaciones"

📜 SEGUNDO TEOREMA

(A · B)̄ = Ā + B̄

"La negación de un producto es igual a la suma de las negaciones"

🛠️ APLICACIÓN DE LOS TEOREMAS

📝 Ejemplo paso a paso:

Expresión original: (A + B) · C̄ + (A + B)̄
Aplicar De Morgan: (A + B) · C̄ + Ā · B̄
Factorizar: (A + B) · C̄ + Ā · B̄
Resultado final: Forma simplificada

⚖️ CIRCUITOS EQUIVALENTES

Diferentes implementaciones de la misma función lógica

🔧 CIRCUITO ORIGINAL

Y = (A + B) · (C + D)

Requiere: 2 OR + 1 AND

Total: 3 compuertas

⚡ CIRCUITO OPTIMIZADO

Y = A·C + A·D + B·C + B·D

Requiere: 4 AND + 1 OR

Total: 5 compuertas
