💡 Clippy Tip: Análisis vs Diseño

ANÁLISIS: Circuito existente → Función lógica

DISEÑO: Especificación → Circuito óptimo

En análisis partimos del hardware hacia la función. En diseño hacemos el camino inverso.

🔍 PROCESO DE ANÁLISIS SISTEMÁTICO

PASO 1 - Inventario: Listar todas las entradas (A, B, C...) y identificar la salida (Y, F...)
PASO 2 - Trazado: Seguir el flujo desde entradas hasta salida, anotando cada operación
PASO 3 - Expresión: Escribir la función booleana completa F = ...
PASO 4 - Tabla: Evaluar todas las combinaciones posibles (2ⁿ filas)
PASO 5 - Verificación: Comprobar casos críticos manualmente
🎯 Clippy Tip: Estrategia de Simplificación

Orden recomendado:

1. Leyes básicas (identidad, complemento)

2. Factorización y distributividad

3. Absorción y redundancia

4. De Morgan si hay negaciones complejas

Siempre verifica: ¿El resultado es más simple que el original?

4. Salida final: Combinar resultados
5. Verificar: Casos especiales y patrones

🎯 Ejercicio para Clase 1

Construir tabla de verdad para: F = A̅BC + AB̅C + ABC̅ + ABC

Pregunta: ¿Cuántos mintérminos tiene esta función?

Tiempo: 8 minutos

⚡ SIMPLIFICACIÓN ALGEBRAICA AVANZADA

Expresión original: F = AB + A̅B + AB̅
Agrupación por factores: F = B(A + A̅) + AB̅
Ley del complemento: F = B(1) + AB̅ = B + AB̅
Ley de absorción: F = B + AB̅ = B + AB̅
Distributividad: F = B + A(B̅) = A + B
Resultado final: F = A + B (¡Mucho más simple!)

🎯 Estrategias de Simplificación

  • Factorización: Buscar términos comunes
  • Absorción: A + AB = A
  • Consenso: AB + A̅C + BC = AB + A̅C
  • Adyacencia: AB + AB̅ = A

⚠️ Verificación Obligatoria

Original: 3 términos, 6 literales
Simplificado: 2 términos, 2 literales
Ahorro: 67% menos compuertas

Siempre verificar con tabla de verdad que ambas expresiones son equivalentes

🎯 Reglas de Agrupación

  • Tamaños válidos: 1, 2, 4, 8 celdas
  • Forma: Rectángulos o cuadrados únicamente
  • Adyacencia: Incluye bordes "envolventes"
  • Objetivo: Grupos más grandes posibles
  • Cobertura: Todos los 1s deben estar cubiertos

📝 Proceso de Minimización:

1. Llenar el mapa: Ubicar 1s según mintérminos de la función
2. Identificar grupos: Formar rectángulos de tamaño potencia de 2
3. Escribir términos: Para cada grupo, eliminar variables que cambian
4. Función final: Sumar todos los términos de grupo (OR)

🗺️ MAPAS DE KARNAUGH - 4 VARIABLES

📊 Estructura 4x4

Para 4 variables (A, B, C, D) necesitamos un mapa 4x4 con 16 celdas.

Variables: AB (filas) × CD (columnas)
Orden Gray en ambas dimensiones
  • Filas: AB̅, AB, AB, AB̅
  • Columnas: CD̅, CD, CD, CD̅
  • Envolvente: Bordes superior-inferior y izquierdo-derecho conectados

🎯 Grupos Especiales en 4 Variables

  • Grupo de 1: 4 literales (ej: ABCD)
  • Grupo de 2: 3 literales (ej: ABC)
  • Grupo de 4: 2 literales (ej: AB)
  • Grupo de 8: 1 literal (ej: A)
  • Grupo de 16: Función = 1 siempre
Regla: Eliminar variables que cambian dentro del grupo
Ubicación de 1s: Mintérminos 0,1,2,5,8,9,10 en el mapa 4x4
Grupo 1: Celdas 0,1,8,9 → Eliminamos variables B y C → AD̅
Grupo 2: Celdas 0,2 → Eliminamos variables A y D → B̅C̅
Grupo 3: Celda 5 individual → AB̅CD
Resultado: F = AD̅ + B̅C̅ + AB̅CD

🎯 Ejercicio Práctico

Pregunta adicional: ¿Esta función tiene algún patrón especial?

⚠️ CASOS ESPECIALES Y CONDICIONES DON'T CARE

❌ Condiciones Don't Care (X)

Combinaciones de entrada que nunca ocurren en el sistema real.

F(A,B,C) = Σ(0,2,6) + d(1,3,5)
  • Ventaja: Mayor flexibilidad de agrupamiento
  • Uso: Pueden ser 0 o 1 según convenga
  • Representación: X o d en el mapa

🔄 Funciones Incompletamente Especificadas

  • BCD natural: Solo códigos 0-9 válidos
  • Control de displays: Algunas combinaciones prohibidas
  • Sistemas de seguridad: Estados imposibles
Las X pueden convertirse en 0 o 1
